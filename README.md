# Sentient AI Chatbot

This project contains the code for a Next.js 15 application written in TypeScript.

## Setup

1. **Extract the project files**
   If the `app/` directory is missing, unzip `sentient-ai-chatbot.zip`:
   ```bash
   unzip sentient-ai-chatbot.zip
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start the development server**
   ```bash
   pnpm dev
   ```

4. **Create a production build**
   ```bash
   pnpm build
   pnpm start
   ```

---
The application uses experimental server actions. If you encounter issues with `next.config.mjs`, update the configuration to match your Next.js version.
